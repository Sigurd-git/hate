\section{研究结果}
\label{sec:results}

本章分别报告研究一(a)与研究一(b)的主要发现。前者刻画九个当代大语言模型（LLM）在中英文仇恨言论识别任务上的总体表现；后者在严格配对设计下检验这些模型对"被指涉对象为女性"的语句是否系统性地给出更高的攻击性评分。两个子研究共用相同的模型组合与提示策略集合（zero-shot 与 chain-of-thought，以下简称 CoT），但评估目标互不重叠：1a 关注分类准确度与概率校准，1b 关注同一语义框架下的性别不对称。

\subsection{研究一(a)：九个 LLM 的中英文仇恨识别表现}
\label{subsec:results-1a}

\subsubsection{两个竞争假设}

在正式呈现结果之前，需先区分两类可被现有数据证伪的假设。
\textbf{H1——任务定义分歧假设：}若模型表现的跨语言差距主要来自中英标注规范与文化边界的分歧，则这一差距对所有 LLM 应\emph{近似为常数}——两个语言面板下九个模型的 F1/Brier 分布仅发生整体平移，\emph{跨模型方差应大致相当}，模型间的相对排序亦应保持稳定。
\textbf{H2——语言覆盖不均假设：}若差距主要源自模型训练语料中的语言结构不均衡，则由于各 LLM 对中文语料的暴露量差异远大于对英文的差异（英文语料几乎覆盖所有厂商的预训练数据，而中文覆盖量在厂商之间差异悬殊），\emph{中文面板的跨模型方差应显著大于英文面板}。换言之，H2 的诊断性特征不是"谁排第一换了"，而是"中文上的模型之间被拉得更开"。
两种假设的可区分预测分别是\emph{等方差}与\emph{方差膨胀}，下文以此为主要判据；模型层面的极端个案（如 Gemma-4-31B 在中文上的塌陷、Qwen-2.5-72B 的反向模式）则作为对主要证据的机制性补充。

\textbf{两个判决性事实。}其一，\emph{均值侧}仅出现温和的整体差距：中文相对英文在 F1 上平均降低约 0.05（zero-shot：$M=-0.046$，$SD=0.069$，7/9 为负；CoT：$M=-0.050$，$SD=0.063$，7/9 为负），在 Brier 上平均升高约 0.04（zero-shot：$M=+0.036$，$SD=0.034$，8/9 为正；CoT：$M=+0.043$，$SD=0.032$，8/9 为正）。若仅看该均值差距，H1 与 H2 均可近似拟合。其二，\emph{方差侧}呈现极端的不对称：中文面板下九个模型的 F1 离散度约为英文面板的 $12.7$ 倍（zero-shot）至 $15.0$ 倍（CoT），Brier 方差比为 $10.7$ 至 $15.1$ 倍，Bartlett 与 Levene 检验均一致拒绝等方差的原假设（见表~\ref{tab:1a-cross-lang}）。换言之，\emph{中文不是把九个模型统一下调，而是把它们拉得更开}——这正是 H2 而非 H1 的诊断性特征。

进一步地，跨语言排序在 F1 上仍相当稳定（$\rho_s=+0.88$ 与 $+0.75$），但在 Brier 上基本失稳（$\rho_s=+0.38$ 与 $+0.28$，均不显著）。这一组合说明：尽管"谁是最强的几个模型"在中英两侧大体一致，中文面板所带来的额外方差主要表现在校准维度——不同模型对中文语料的概率校准差异远大于英文。这与"中文训练语料在厂商之间的暴露量差异显著大于英文"这一 H2 所依赖的经验前提相吻合，也解释了为何两类指标在统计上表现出并不完全平行的跨语言行为。

\subsubsection{方差膨胀的模型层面来源}

上一小节以面板层面的统计确立了"中文方差显著大于英文"这一主要现象。本小节进一步追问：该方差膨胀由哪些模型驱动？图~\ref{fig:1a-f1-bars}与图~\ref{fig:1a-f1-dumbbell}分别给出四个面板下的 F1 条形与跨提示策略的哑铃图，便于逐模型比较。

\textbf{中文面板的 F1 极差}在 zero-shot 下达到 $0.289$（Qwen $0.751$ 对 Gemma-4-31B $0.463$），在 CoT 下为 $0.264$（Llama $0.736$ 对 Gemma $0.472$）；英文面板的同尺度极差仅为 $0.074$（zero-shot）与 $0.063$（CoT）。该额外方差不是九个模型等量贡献，而主要由以下三类个案累积而成。

第一，\textbf{Gemma-4-31B 在中文上的系统性塌陷}构成下端离群值：该模型在英文两个面板下均维持在 $0.63$ 附近，而在中文两个面板中骤降至 $0.47$ 附近（$\Delta_{zh-en}=-0.171$ 于 zero-shot，$-0.164$ 于 CoT，为九个模型中最极端的负差）。H1 的"任务定义分歧"框架无法解释为何这一定义差距对单一模型的影响远超对其他模型的影响。

第二，\textbf{Qwen-2.5-72B 在中文上的反向优势}构成上端离群值：九个模型中仅该模型在 zero-shot 下出现 $\Delta_{zh-en}>0$（$+0.066$），且其绝对 F1 值 $0.751$ 是全体数据中的最大值。这两端的反向拉动——Gemma 向下、Qwen 向上——即为中文面板方差膨胀的直接几何来源。

第三，\textbf{CoT 并非均质地压缩方差}。Qwen 在中文 zero-shot 下 F1 为 $0.751$，切换到 CoT 反而降至 $0.639$；Llama 在中文 zero-shot 下为 $0.641$，在 CoT 下升至 $0.736$。两个面板的方差比因而从 $12.7$ 倍略升至 $15.0$ 倍——提示策略的切换并未让九个模型收敛到同一难度基线，反而重新分配了哪些模型占据中文面板的上端。若 H1 成立，作为通用推理放大手段的 CoT 应当带来稳定且近似均质的增益；图~\ref{fig:1a-f1-dumbbell}呈现的异质方向进一步削弱这一叙事。

\textbf{厂商来源的弱预测力。}一个更直接的 H2 机制性假设是：由中文厂商训练的 LLM 接触到更多中文预训练语料，因此在中文面板上平均表现更好。对 9 个模型按注册地粗分（中文厂商 $n=5$：Qwen、GLM、Kimi、DeepSeek R1、DeepSeek V3.2；西方厂商 $n=4$：GPT-5.1、Claude 4.5、Llama-4-Maverick、Gemma-4-31B），zero-shot 下两组的平均 $\Delta_{zh-en}$ 分别为 $-0.030$ 与 $-0.065$；但剔除 Gemma 后，西方厂商组的均值回落至 $-0.030$，与中文厂商组持平。CoT 下两组均值更为接近（$-0.049$ vs.\ $-0.051$）。这意味着"厂商来源"本身只能说明\emph{方差的部分来源}（Gemma 作为极端个案）而非系统性的均值差异；九个模型内部对中文语料的有效暴露量显然存在一个更细粒度的、超越注册地的分布，这也正是为什么下一小节将把对 1a 的结论限制在"方差膨胀存在且与 H2 一致"，而不进一步强诉单一的机制性解释。

综合上述三类个案与厂商对比，我们将本节观察解读为对 H2 的支持：中英方差不对称由中文一侧的模型层面极值驱动，而这些极值与"模型对中文语料的非均匀暴露"这一前提相容。\emph{完全驳回 H1 尚需跨标注体系的证据}（例如在同一套边界定义下重新打标的中英平行集），但当前数据所表现的方差膨胀形态已超出 H1 所允许的"整体平移"预测。

\subsubsection{校准（Brier）与准确率（F1）的解耦}

F1 与 Brier 度量的是两种不同的能力——"能否将句子判为正类"与"能否给出与真实频率一致的概率"。若两者高度耦合，则仅报告 F1 即可；反之则需分别考察。图~\ref{fig:1a-brier-bars}与图~\ref{fig:1a-brier-dumbbell}呈现 Brier 结果。

\textbf{解耦的关键证据}出现在英文 zero-shot 面板：F1 最高者为 Llama-4-Maverick（0.698），但 Brier 最低者为 Qwen-2.5-72B（0.254），且 Llama 的 Brier 反而较高（0.297）。换言之，Llama 更常给出\emph{方向正确但过度自信}的预测，而 Qwen 的预测虽分类略逊，却在概率尺度上更贴近真实频率。这说明若下游任务需要可靠的风险阈值（例如人机协作审核中的"模糊样本留待人工判定"），F1 榜单的第一名并非最佳选择。

此外，CoT 对 Brier 的影响呈现\emph{非单调}形态：对 Llama 在中文上显著降低 Brier（从 zero-shot 的 0.300 降至 CoT 的 0.243），却对 Qwen 在英文上反而恶化（从 0.254 升至 0.289）。这一交互进一步削弱"CoT 通用增益"叙事。

\subsubsection{自然数据下对"针对女性"与"针对男性"的评分：样本构成的混淆}
\label{subsubsec:1a-gender-natural}

总体 F1／Brier 掩盖了一个与研究一(b) 直接相关的问题——\emph{LLM 对针对女性的仇恨帖与针对男性的仇恨帖的攻击性打分是否系统性不同}？一个先验合理的预期是：若模型携带"女性方向"的评估偏差，则针对女性的帖子应获得更高的平均攻击分。本小节以自然数据（HateXplain 的 \texttt{target} 字段）直接检验这一预期，不对样本的语义内容做任何配平。这与研究一(b) 的设计形成对照：前者语义内容在女性／男性子集间自由变化，后者通过配对句将非性别语义内容严格固定。

\begin{figure}[htbp]
    \centering
    \includegraphics[width=\linewidth]{../artifacts/placeholder_a_1a_gender_target_mean_score.pdf}
    \caption{九个 LLM 在英文 HateXplain 自然数据上对"针对女性"与"针对男性"帖子的平均攻击打分（0–5 分）。以 \texttt{post\_id} 为键将合并验证集与 HateXplain 原始 \texttt{dataset.json} 做连接，并按各标注者在 \texttt{target} 字段中标注的 Women／Men 决定样本归属：若至少一位标注者标记 \texttt{Women} 且无人标记 \texttt{Men}，该帖归为"针对女性"（$n=175$）；反之归为"针对男性"（$n=36$）；两者兼有或均无者被剔除。中文侧未纳入本图——原始中文语料并无男／女目标字段，基于词汇标记的启发式分类在九个模型上方向分裂、量级接近零，缺乏解释价值，细节见附录。}
    \label{fig:1a-gender-natural}
\end{figure}

\textbf{观察要点：}九个模型在两种 prompt 设置（zero-shot、CoT）下\emph{均}将"针对男性"帖子的平均分打得高于"针对女性"帖子（total: zero-shot $3.19$ vs $2.64$；CoT $3.35$ vs $2.72$），方向与"女性方向偏差"的预期相反。这一反向结果不应被解读为模型保护男性：HateXplain 中 \texttt{Men} 目标样本量仅 36 条，且高度集中在与族群（尤其是针对黑人男性）交叉的极端仇恨语境中；\texttt{Women} 样本则涵盖更广的仇恨强度谱。换言之，\emph{自然数据的女／男子集在语义内容上并非可比}——任何子集间差异都混合了样本构成效应与潜在评估偏差，且前者在此数据上远大于后者。这正是研究一(b) 采用配对设计的直接动因：只有在词汇与语义框架被强制匹配、仅改变性别指涉的前提下，"女性 vs 男性"的差异才能被归因于评估偏差本身。如下一节将展示，在该配对控制下，九个模型\emph{一致}出现女性指涉版评分高于男性指涉版的结果——方向与本节自然数据相反，支持"样本构成混淆了 1a 自然数据中的真实评估偏差"这一解读。

\subsection{研究一(b)：性别群体对换下的攻击性评分}
\label{subsec:results-1b}

\subsubsection{判决性实验：两个竞争假设}

研究一(b)的核心设计是\emph{最小对比对}（minimal pairs）——对同一语义框架下的一个句子仅替换指涉对象的性别代称（例如"女人"$\leftrightarrow$"男人"，"她"$\leftrightarrow$"他"），其余词汇完全一致。这一设计用于区分下述两类假设。
\textbf{H1'——语义内容假设：}模型对攻击性的判定应由句子所指涉的语义内容决定；若配对句的语义框架相同，则两个版本的评分差应以 0 为中心、对称分布。
\textbf{H2'——评估偏差假设：}模型携带一个独立于语义内容的、指向女性方向的攻击性评估偏差；若该偏差存在，则（女性指涉版 − 男性指涉版）的评分差应系统性大于 0。

需强调，H1' 并不要求模型"对所有内容给零分"，仅要求\emph{配对差值的期望为零}。该设计意味着任何来自词频、讨论度或语料中背景分布的混淆因素在配对内部被抵消。H2' 的一个具体预测是，若偏差普遍存在于当代 LLM，则不同模型在方向性与效应量上应呈\emph{一致而非散漫的}结果。

配对语料共 371 对中文句子，涉及 10 个一级攻击领域（level-1 attack domain）、71 个二级子类与 183 个三级标签。主分析以 7 点 Likert 尺度为主（\texttt{attack\_7pt\_likert}），3 点尺度与 0–100 连续滑块尺度将在节~\ref{subsubsec:1b-scale-invariance}中用作稳健性检验。

\subsubsection{同一模型内部的跨尺度偏差模式对齐}
\label{subsubsec:1b-within-model-cross-scale}

前一小节的 $d_z$ 等级稳定性仅证明\emph{聚合量级}跨尺度可比；它不保证\emph{题项层面}的偏差指向在三尺度间是同一个分布。形式上，定义第 $i$ 个配对句在模型 $m$ 与尺度 $s$ 下的题项级偏差为
\[
\Delta_{m,i}^{(s)} = y_{m,i,\text{女}}^{(s)} - y_{m,i,\text{男}}^{(s)},
\]
对每一个模型 $m$，我们得到三条长度为 371 的向量 $\{\Delta_{m,\cdot}^{(\text{3pt})}, \Delta_{m,\cdot}^{(\text{7pt})}, \Delta_{m,\cdot}^{(\text{slider})}\}$，并计算三对向量的 Spearman $\rho$ 与 Kendall $\tau$（95\% CI 由 $B=2000$ 次题项重抽样 bootstrap 得到）。若模型在三尺度下对"\emph{哪一句}偏差更大"这一题项级问题给出的是同一个答案，$\rho$ 应普遍接近 1；若尺度本身改变了模型对题项排序的依据，$\rho$ 应接近 0 甚至显著为负。

\textbf{一句话读懂此表与图~\ref{fig:1b-within-model-cross-scale}：}让同一个 LLM 用三种尺度打同一批 371 对句子，它对"哪一句最偏差"三次给出的答案几乎没有重合——中位 Spearman $\rho$ 只有 $0.006$--$0.15$，远远低于"若尺度只改变分辨率、不改变内容"应有的接近 1 的水平。也就是说：LLM 在"整体偏差量级"上三尺度一致，但在"具体挑哪一句"上三尺度彼此独立。

\textbf{核心发现：题项级偏差模式的跨尺度对齐度显著低于聚合量级的对齐度。}九个模型 $\times$ 三对尺度组合共 27 个单元中，Spearman $\rho$ 的中位数仅为 $0.006$（3pt vs 7pt）、$0.152$（7pt vs slider）与 $0.116$（3pt vs slider）——三者均显著偏离"若尺度只是同一潜在偏差的单调映射则应出现的 $\rho$ 接近 1"这一理论预期。换言之，虽然\emph{平均偏差量级}跨尺度稳定，\emph{哪一句被判为最偏差}却在尺度间发生了系统性重新分配。

该解耦在三条模式层面进一步得以刻画。第一，\emph{3 点尺度在三对组合中都是"离群的那一端"}——与 7 点或滑块比较时，3 点相关性系统性最低（3pt vs 7pt 中位 $\rho=0.006$、3pt vs slider 中位 $\rho=0.116$，均低于 7pt vs slider 的 $0.152$）。该结果与尺度分辨率的解释一致：3 点只能区分三档，$\Delta_i$ 只能取有限几个离散值，题项层面的细粒度排序信息被量化损失。第二，\emph{跨尺度对齐度在模型间高度异质}：GPT-5.1 与 Gemma-4-31B 在三对尺度上 $\rho$ 均为正且相对较高（$\rho \approx 0.23\text{–}0.36$）；而 Claude-4.5 在 3pt vs 7pt 上出现显著的\emph{负}相关（$\rho = -0.233$），其 CI 不含 0——这意味着 Claude 在 3 点下判为"更偏差"的题项，到了 7 点下反而倾向判为"更不偏差"。此异常构成对"偏差是单一潜结构"假设的直接反例。第三，\emph{负向或零相关的模型对"哪一句最偏差"并无稳定内部共识}：DeepSeek-V3.2（3pt vs 7pt $\rho=-0.035$）、Llama-Maverick（3pt vs 7pt $\rho=-0.073$）、Qwen-2.5-72B（3pt vs 7pt $\rho=-0.019$）在 3 点与 7 点的排序几乎独立。

对研究问题"同一模型在三尺度下是否存在差异"，本节给出一个分层回答：\emph{在聚合的 $d_z$ 上几乎没有差异}；\emph{在题项级 $\Delta_i$ 的排序上差异巨大}。方法论上这意味着，研究性别偏差的\emph{题项级特征}（例如"哪一类句子最能激发 LLM 的女性化加权"）时，评分尺度的选择不是一个中性的操作变量——它改变了问题本身。实质性上，它削弱了"LLM 的性别偏差是一个\emph{单一}潜在向量、只是被不同尺度以不同分辨率取样"这一简单表述，暗示偏差更可能是一个\emph{多维}的、在不同答题格式下被不同子过程调取的构念。本发现对后文研究二(a) 的 ICC 分析（节~\ref{subsec:results-2a-icc}）具有直接的方法论意义：若人机相似度在 3 点与 7 点／滑块之间跨尺度不一致，这并非仅是人类侧的异质，也反映了 LLM 侧题项级偏差本身的尺度依赖性。

\subsubsection{模型 \texttimes{} 维度的一致性模式}
\label{subsubsec:1b-model-by-dim}

\subsubsection{尺度稳健性：尺度 $\ll$ 性别}
\label{subsubsec:2a-scale-invariance}

将上节 $2 \times 3$ 网格另行按"固定分组、变动尺度"与"固定尺度、变动分组"两个方向阅读：（a）\emph{固定被试分组、变动尺度}时，$d_z$ 跨尺度差异小：女性被试 $0.655 \to 0.704 \to 0.590$（极差 $0.114$），男性被试 $0.219 \to 0.198 \to 0.167$（极差 $0.052$）；（b）\emph{固定尺度、变动分组}时，$d_z$ 在每个尺度下相差约 $0.43$--$0.51$。后者的组间差大约是前者组内极差的 4--8 倍。换言之，人类被试的偏差\emph{不依赖}于评分尺度，但\emph{显著依赖}于被试性别；尺度选择是一个相对中性的测量决策，而被试性别是决定效应量的关键人群学变量。

\subsection{研究二(a)：人机偏差模式的题项级相似度}
\label{subsec:results-2a-icc}

\subsubsection{研究问题与统计设计}

研究一(b) 与节~\ref{subsec:results-2a}均表明：LLM 与人类被试在方向与量级上对同质配对语句表现出一致的性别偏差。但"方向一致"不等于"模式一致"——还必须问：\emph{在题项层面上，LLM 的偏差分布与人类的偏差分布有多接近？哪一句被判为偏差最大这一问题上，两者是否达成了一致？}

对每一题项 $i$ 定义人机两端的题项级偏差为：
\[
\Delta_i^{\text{人}} \;=\; \bar y_{i,\text{女}}^{\text{人}} - \bar y_{i,\text{男}}^{\text{人}}, \qquad
\Delta_i^{\text{模}} \;=\; y_{i,\text{女}}^{\text{模}} - y_{i,\text{男}}^{\text{模}},
\]
其中人类端为所给尺度下各被试的题项均值。以\emph{题项级相似度}作为"偏差模式相像"的操作定义，主指标为 ICC(2,1) 绝对一致（single rater, two-way random），对照指标为 ICC(3,1) 一致性（two-way mixed, single rater）；前者同时惩罚排序偏离与系统性平移，后者只惩罚排序偏离。二者均以 bootstrap（$B=2000$，按题项重抽样）给出 95\% CI。按尺度特性另补报 robustness metrics：3 点下的 quadratic-weighted Cohen's $\kappa$（将 $\Delta_i$ 按 $\{-1, 0, +1\}$ 符号三分类）、7 点下的 Spearman $\rho$、滑块下的 Lin's concordance correlation coefficient 与 RMSE／MAE。

\textbf{关键方法论约束}：相似度计算\emph{严格限定在同一尺度内进行}——人类 3 点 $\Delta_i$ 仅与 9 个 LLM 的 3 点 $\Delta_i$ 比较；7 点同理；滑块同理。此约束有两个论据支撑：其一，节~\ref{subsubsec:1b-within-model-cross-scale}已直接证明同一 LLM 在三尺度间的题项级偏差向量并不共享一个单一潜结构（Spearman $\rho$ 中位数仅 $0.006$--$0.152$），因此跨尺度对比会把尺度格式效应与偏差效应混淆；其二，人类侧的三尺度 $\Delta_i^{\text{人}}$ 同样存在尺度格式差异，跨尺度比较意义有限。此外，人类端分三层（全体被试、男性被试、女性被试）独立计算 $\Delta_i^{\text{人}}$，得到 $3 \times 3 = 9$ 个"被试子群 $\times$ 尺度"单元，每单元遍历 9 个 LLM，选出 ICC(2,1) 最高者为该单元"最相似模型"。

\subsubsection{三条可报告的观察}

\textbf{观察一：最相似模型随被试子群漂移——"谁最像人"依赖于"人是谁"。}9 个单元下，"最像人类"的 LLM 分布在 5 个不同模型之间：Claude Opus 4.5（3 次）、Llama 4 Maverick（2 次）、DeepSeek R1（2 次）、GLM 4.6（1 次）、Qwen 2.5 72B（1 次）。全体被试样本下 Claude 4.5 在 7 点与滑块两列同时胜出；但拆分到被试性别后——7 点：男性被试对应 GLM 4.6、女性被试对应 Claude 4.5；滑块：男性被试对应 DeepSeek R1、女性被试对应 Qwen 2.5 72B；仅 3 点的最相似模型在全体与男性被试下重合（均为 Llama 4 Maverick）。这一漂移模式的含义是：\emph{全体样本的"最像人类"结论是对"被试性别"这一维度的隐式平均}；任何关于"某个 LLM 的偏差最接近人类直觉"的声明，都必须显式交代其所基于的被试子群与评分尺度。

\textbf{观察二：ICC 水平随尺度分辨率单调上升——题项级信号在 3 点下被压缩殆尽。}全体被试下最大 ICC(2,1) 从 3 点的 $0.089$ 升至 7 点的 $0.158$ 再升至滑块的 $0.222$；男性被试下为 $0.096 \to 0.155 \to 0.174$；女性被试下为 $0.100 \to 0.116 \to 0.112$。按 Koo \& Li (2016) 阈值体系（$<0.50$ 为 poor、$0.50$--$0.75$ 为 moderate），全部 9 个单元均位于 poor 区间——即便是该单元下"最接近人类"的模型，其相似度绝对值仍然较低。该现象不是"LLM 在连续尺度上更像人"的实质证据，而是"3 点尺度将题项级 $\Delta_i$ 压缩至至多三档取值后可检测的对齐信号显著衰减"的直接结果。该发现与节~\ref{subsubsec:1b-within-model-cross-scale}（LLM 内部跨尺度比较，3 点同样是三尺度中最"离群"的一端）在两条独立证据链上相互印证——\emph{3 点尺度在题项级偏差研究中具有系统性的信号丢失}，后续题项级对齐研究建议以 7 点或滑块为主尺度。

\textbf{观察三：男性被试组的 ICC 信号系统性强于女性被试组——与 $d_z$ 方向相反的"量级 $\text{-}$ 模式"张力。}男性被试在 7 点（$0.155$，CI $[0.056, 0.256]$）与滑块（$0.174$，CI $[0.081, 0.256]$）下 CI 均明确不含 0；女性被试在滑块下 CI 跨越 0（$[-0.008, 0.223]$）、7 点下勉强不含 0（$[0.010, 0.219]$）。此方向与节~\ref{subsubsec:2a-participant-gender}的"女性被试 $d_z$ 是男性被试 3 倍"形成了明显张力——\emph{效应量更大的被试组反而与 LLM 的题项级偏差模式对齐更弱}。该张力表明两套指标刻画的并非同一维度的"偏差"：$d_z$ 捕获的是\emph{偏差的总体量级}，ICC 捕获的是\emph{偏差在题项上的分配}。换言之，女性被试在整体上"更倾向判女人版为攻击"，但\emph{她们所偏向的具体题项}与 9 个 LLM 所偏向的\emph{具体题项}重合度较低；男性被试总体量级较弱，但其偏向的少量题项与 LLM（尤其 DeepSeek R1、GLM 4.6）更一致。

该张力可以被表述为一个具体的、可后续检验的机制性假设：\emph{当前 9 个 LLM 的对齐过程更多捕获了"被男性评分者识别为攻击性"的题项特征，而尚未充分对齐女性评分者独有的题项级敏感性}——该差异的本质即是"对齐偏斜"（alignment skew）而非"对齐不足"（alignment shortfall）。该假设可通过"以女性评分者为主体的偏好数据做 RLHF 再校准"的后续实验加以直接检验。

\subsection{研究二(a)：打分尺度对题项级偏差模式的影响——人类 vs.\ LLM 的稳健性对比}
\label{subsec:results-2a-cross-scale}

\subsubsection{研究问题}

节~\ref{subsubsec:1b-within-model-cross-scale}已对九个 LLM 作过"同一模型在三个尺度下题项级 $\Delta_i$ 是否相互对齐"的分析。一个自然的对照问题是：\emph{人类被试在三个打分尺度下的题项级偏差模式是否也存在这种尺度依赖性？若然，与 LLM 相比，谁更受打分尺度的影响？}若假设人类评分较易受题面呈现方式、记忆负担与锚点认知等格式性因素影响，则人类的跨尺度 $\rho$ 应偏低；若假设 LLM 作为"纯语言生成器"能在不同打分尺度下对同一内容给出成比例的评分（例如对同一条句子，3 点给 2、7 点给 $4$--$5$、滑块给 $66$ 附近），则 LLM 的跨尺度 $\rho$ 应显著高于人类。两种假设在本节得到直接对照。

\subsubsection{方法}

对每一被试子群（全体被试／男性被试／女性被试），计算该子群在三尺度下的 $\Delta_i^{\text{人}}$ 向量（长度 371），然后对三对尺度组合（3pt vs 7pt、7pt vs slider、3pt vs slider）分别计算 $\Delta_i^{\text{人}}$ 的 Spearman $\rho$，并以 bootstrap（$B=2000$，按题项重抽样）给出 95\% CI。LLM 端复用节~\ref{subsubsec:1b-within-model-cross-scale}的 $\Delta_i^{\text{模}}$ 跨尺度 $\rho$。所有人机比较严格限定在\emph{同一对尺度组合内}进行——即人类的"3pt 与 7pt 对齐度"仅与 LLM 的"3pt 与 7pt 对齐度"比较，其它尺度对同理。

\subsubsection{三条诚实的发现}
\label{subsubsec:2a-cross-scale-honest-findings}

本节将"我们原本怎么猜"和"数据实际怎么说"逐条对照，把对先验假设的验证结果写成易读结论。

\textbf{发现一：人类在三种打分尺度下"偏差模式都不一样"——假设得到支持。}我们事先预期：即使是同一批人类被试、打同一批 371 对句子，只要换一种评分尺度（3 点 / 7 点 / 滑块），"哪一句最偏"的答案就会改变。数据直接证实了这一预期——\emph{3 个被试子群} $\times$ \emph{3 对尺度组合} 共 9 个单元里，跨尺度 Spearman $\rho$ 的中位数仅为 $0.105$（3pt vs 7pt）、$0.077$（7pt vs slider）、$0.108$（3pt vs slider），远低于"若尺度只是改变分辨率"应有的接近 1 的水平；绝对值 9 个单元都 $<0.13$，多数单元的 bootstrap 95\% CI 跨越 0。人类的题项级偏差模式在三种尺度下几乎是三套独立的答案。

\textbf{发现二："3 点与 7 点更相似、二者都跟滑块差最远"的推测——没有数据支持。}按先验推测，3 点与 7 点都是离散 Likert，"同家族"应当相似；连续滑块是另一族，应当差最远。若该推测成立，我们应当在三对组合上看到 $\rho_{\text{3pt-7pt}} \gg \rho_{\text{7pt-slider}} \approx \rho_{\text{3pt-slider}}$ 的有序模式。实际数据却是三对组合的中位 $\rho$ 相互接近（$0.105 / 0.077 / 0.108$），三个被试子群内部的大小排序也不一致（全体被试：$0.118 > 0.124 > 0.091$；男性：$0.105 > 0.066 > 0.008$；女性：$0.108 > 0.077 > 0.011$）。换言之，\emph{只要换尺度就是全方位的重组}——不是"连续 vs 离散"这一二分决定差异，3 档与 7 档的分辨率差已足以把题项级排序打乱。

\textbf{发现三："LLM 比人更理性客观、跨尺度应更稳定"——也没有数据支持。}若 LLM 在不同尺度下共享同一把"偏差尺子"，那它对同一条句子的 $\Delta_i$ 在三种尺度上应当成比例（如 3 点给 2、7 点给 $4$--$5$、滑块给 $66$ 附近）。但数据显示：LLM 的跨尺度 $\rho$ 中位数（$0.006 / 0.152 / 0.116$）与人类中位数（$0.105 / 0.077 / 0.108$）处于同一水平；在 3pt vs 7pt 这一对上，LLM 中位数 $0.006$ 甚至\emph{低于}人类 $0.105$——即 LLM 并没有在这对尺度间给出比人类更对齐的题项级偏差。

真正的人机差异不在"平均谁更稳健"，而在\emph{离差}。3 个人类子群的跨尺度 $\rho$ 相互接近（全部落在 $[0.008, 0.124]$ 区间，跨度仅 $0.11$）；9 个 LLM 的跨尺度 $\rho$ 则从负到正横跨约 $0.5$ 个单位（例如 3pt vs 7pt 上 Claude Opus 4.5 为 $-0.233$、GPT-5.1 为 $+0.293$，跨度 $0.526$，约为人类跨度的 5 倍；7pt vs slider、3pt vs slider 上的跨度比也分别为 4.5 倍与 4.7 倍）。换言之，\emph{LLM 不是整体上更稳健或更不稳健，而是更两极分化}——GPT-5.1 与 Gemma-4-31B 三对 $\rho$ 一致为正、$0.23$--$0.36$，确实比人类更跨尺度稳健；而 Claude Opus 4.5 在 3pt vs 7pt 上出现显著负 $\rho$（CI 不含 0），表明该模型在两个尺度下使用了\emph{实质不同}的评分策略。每个 LLM 都有自己一套跨尺度行为。

\subsubsection{可报告的综合结论}
\label{subsubsec:2a-cross-scale-conclusion}

用一句话概括：\emph{"题项级偏差在不同打分尺度下是同一套偏差被格式化地不同采样"这一朴素图景，在人类与 LLM 两端均不成立}。人类的违反是 9 个单元下中位水平的低 $\rho$，模式一致但全面偏低；LLM 的违反则是高异质的——某些模型（GPT-5.1、Gemma-4-31B）确实比人类更跨尺度稳健，某些模型（Claude Opus 4.5、Llama 4 Maverick）显著不稳健。

这一发现对本章的 ICC 题项级对齐分析（节~\ref{subsec:results-2a-icc}）有一个可立即使用的方法论推论：即便未来人类评分样本量显著增加、$\Delta_i^{\text{人}}$ 的噪声显著下降，\emph{3 点尺度下的人机 ICC 仍将受限}——因为不仅是人类侧的噪声，LLM 侧本身也没有给出一个与 7 点／滑块共享的、稳健的 3 点 $\Delta_i^{\text{模}}$ 结构（节~\ref{subsubsec:1b-within-model-cross-scale}）。因此"在 3 点尺度上做题项级人机偏差对齐研究"这一范式无法被单靠样本量扩张拯救；若研究目的是量化题项级对齐，应当把主分析锚定在 7 点或滑块尺度上，3 点仅作稳健性对照。

\subsection{研究范围与分析边界}
\label{subsec:scope-notes}

上文研究一(b) 与研究二(a) 的配对分析当前仅在\emph{中文}语境下展开：371 对配对句与对应的三级分类体系均为中文自建语料，9 个 LLM 被评估于该套中文配对，712 名人类被试亦为中文母语者。跨语言泛化所需的英文配对语料目前尚未完成，本文不对"评估偏差假设"在英文条件下的表现做出声明；该方向的延伸研究将在讨论部分作为未来工作列出。此外，题项级人机相似度分析（节~\ref{subsec:results-2a-icc}）的 ICC 绝对值均落在 Koo \& Li (2016) 意义下的 poor 区间，这并非实验失败，而是"人机在题项级偏差排序上仅存在弱对齐"这一真实现象的度量结果——该观察本身即构成一个可报告的发现。
