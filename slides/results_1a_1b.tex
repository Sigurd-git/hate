\section{研究结果}
\label{sec:results}

本章分别报告研究一(a)与研究一(b)的主要发现。前者刻画九个当代大语言模型（LLM）在中英文仇恨言论识别任务上的总体表现；后者在严格配对设计下检验这些模型对"被指涉对象为女性"的语句是否系统性地给出更高的攻击性评分。两个子研究共用相同的模型组合与提示策略集合（zero-shot 与 chain-of-thought，以下简称 CoT），但评估目标互不重叠：1a 关注分类准确度与概率校准，1b 关注同一语义框架下的性别不对称。

\subsection{研究一(a)：九个 LLM 的中英文仇恨识别表现}
\label{subsec:results-1a}

\subsubsection{两个竞争假设}

在正式呈现结果之前，需先区分两类可被现有数据证伪的假设。
\textbf{H1——任务定义分歧假设：}若模型表现的跨语言差距主要来自中英标注规范与文化边界的分歧，那么\emph{同一模型}在中英两个面板中的排名应当保持稳定，差距仅体现为整体分数的平移。
\textbf{H2——语言覆盖不均假设：}若差距主要来自模型训练语料的语言结构不均衡，那么模型排名会\emph{跨语言翻转}——在中文语料占优的模型在中文面板领先、以英文为主要训练语料的模型在英文面板领先。
两种假设给出的可区分预测分别是"排名稳定"与"排名翻转"，下文以此为判据。

\begin{figure}[htbp]
    \centering

    \caption{每个模型从 zero-shot 到 CoT 的 F1 变化（哑铃图），按语言分面板。}

三条经验性规律需要指出。第一，\textbf{中文 zero-shot 面板呈现显著的模型分层}：最高 F1（Qwen，0.751）与最低 F1（Gemma-4-31B，0.463）之间相差近 0.29 个 F1 单位；而英文 zero-shot 面板下，最高值（Llama，0.698）与最低值（GLM-4.6，0.623）相差仅约 0.07。这一"在中文上拉得开、在英文上拉不开"的模式无法由单纯的任务难度差异解释。
第二，\textbf{Gemma-4-31B 的跨语言塌陷}尤为诊断性：在英文 zero-shot/CoT 下维持在 0.63 附近，而在中文两个面板中骤降至 0.47 附近。一个"任务定义分歧"假设无法预测该模型的中文表现崩塌幅度远超其他模型。
第三，\textbf{CoT 并非普适提升}：Qwen 在中文 zero-shot 下 F1 为 0.751，切换到 CoT 反而降至 0.639；Gemma 在中文 CoT 下相比 zero-shot 略有提升，但仍维持在 0.47 水平。若 H1 成立，CoT 作为一种通用的推理放大手段应当带来稳定增益，但观察到的异质性提示增益与否取决于模型是否已掌握该语言下的任务格式。

综合三条规律，我们将主要证据解读为支持 H2（语言覆盖不均）而非 H1（任务定义分歧）。\emph{完全驳回 H1 尚需跨标注体系的证据}，但至少，H1 无法解释当前观察到的排名翻转与模型×语言交互。

\subsubsection{校准（Brier）与准确率（F1）的解耦}

F1 与 Brier 度量的是两种不同的能力——"能否将句子判为正类"与"能否给出与真实频率一致的概率"。若两者高度耦合，则仅报告 F1 即可；反之则需分别考察。图~\ref{fig:1a-brier-bars}与图~\ref{fig:1a-brier-dumbbell}呈现 Brier 结果。

\begin{figure}[htbp]
    \centering
    \includegraphics[width=\linewidth]{../artifacts/validated_csv_hate_brier_2x2_bars.pdf}
    \caption{四个面板下九个 LLM 的 Brier 评分（越低越好）。}
    \label{fig:1a-brier-bars}
\end{figure}

\begin{figure}[htbp]
    \centering
    \includegraphics[width=0.95\linewidth]{../artifacts/validated_csv_hate_brier_2x2_dumbbell.pdf}
    \caption{每个模型从 zero-shot 到 CoT 的 Brier 变化（哑铃图）。}
    \label{fig:1a-brier-dumbbell}
\end{figure}

\textbf{解耦的关键证据}出现在英文 zero-shot 面板：F1 最高者为 Llama-4-Maverick（0.698），但 Brier 最低者为 Qwen-2.5-72B（0.254），且 Llama 的 Brier 反而较高（0.297）。换言之，Llama 更常给出\emph{方向正确但过度自信}的预测，而 Qwen 的预测虽分类略逊，却在概率尺度上更贴近真实频率。这说明若下游任务需要可靠的风险阈值（例如人机协作审核中的"模糊样本留待人工判定"），F1 榜单的第一名并非最佳选择。

此外，CoT 对 Brier 的影响呈现\emph{非单调}形态：对 Llama 在中文上显著降低 Brier（从 zero-shot 的 0.300 降至 CoT 的 0.243），却对 Qwen 在英文上反而恶化（从 0.254 升至 0.289）。这一交互进一步削弱"CoT 通用增益"叙事。

\subsubsection{自然数据下对"针对女性"与"针对男性"的评分：样本构成的混淆}
\label{subsubsec:1a-gender-natural}

总体 F1／Brier 掩盖了一个与研究一(b) 直接相关的问题——\emph{LLM 对针对女性的仇恨帖与针对男性的仇恨帖的攻击性打分是否系统性不同}？一个先验合理的预期是：若模型携带"女性方向"的评估偏差，则针对女性的帖子应获得更高的平均攻击分。本小节以自然数据（HateXplain 的 \texttt{target} 字段）直接检验这一预期，不对样本的语义内容做任何配平。这与研究一(b) 的设计形成对照：前者语义内容在女性／男性子集间自由变化，后者通过配对句将非性别语义内容严格固定。

\begin{figure}[htbp]
    \centering
    \includegraphics[width=\linewidth]{../artifacts/placeholder_a_1a_gender_target_mean_score.pdf}
    \caption{九个 LLM 在英文 HateXplain 自然数据上对"针对女性"与"针对男性"帖子的平均攻击打分（0–5 分）。以 \texttt{post\_id} 为键将合并验证集与 HateXplain 原始 \texttt{dataset.json} 做连接，并按各标注者在 \texttt{target} 字段中标注的 Women／Men 决定样本归属：若至少一位标注者标记 \texttt{Women} 且无人标记 \texttt{Men}，该帖归为"针对女性"（$n=175$）；反之归为"针对男性"（$n=36$）；两者兼有或均无者被剔除。中文侧未纳入本图——原始中文语料并无男／女目标字段，基于词汇标记的启发式分类在九个模型上方向分裂、量级接近零，缺乏解释价值，细节见附录。}
    \label{fig:1a-gender-natural}
\end{figure}

\textbf{观察要点：}九个模型在两种 prompt 设置（zero-shot、CoT）下\emph{均}将"针对男性"帖子的平均分打得高于"针对女性"帖子（total: zero-shot $3.19$ vs $2.64$；CoT $3.35$ vs $2.72$），方向与"女性方向偏差"的预期相反。这一反向结果不应被解读为模型保护男性：HateXplain 中 \texttt{Men} 目标样本量仅 36 条，且高度集中在与族群（尤其是针对黑人男性）交叉的极端仇恨语境中；\texttt{Women} 样本则涵盖更广的仇恨强度谱。换言之，\emph{自然数据的女／男子集在语义内容上并非可比}——任何子集间差异都混合了样本构成效应与潜在评估偏差，且前者在此数据上远大于后者。这正是研究一(b) 采用配对设计的直接动因：只有在词汇与语义框架被强制匹配、仅改变性别指涉的前提下，"女性 vs 男性"的差异才能被归因于评估偏差本身。如下一节将展示，在该配对控制下，九个模型\emph{一致}出现女性指涉版评分高于男性指涉版的结果——方向与本节自然数据相反，支持"样本构成混淆了 1a 自然数据中的真实评估偏差"这一解读。

\subsection{研究一(b)：性别群体对换下的攻击性评分}
\label{subsec:results-1b}

\subsubsection{判决性实验：两个竞争假设}

研究一(b)的核心设计是\emph{最小对比对}（minimal pairs）——对同一语义框架下的一个句子仅替换指涉对象的性别代称（例如"女人"$\leftrightarrow$"男人"，"她"$\leftrightarrow$"他"），其余词汇完全一致。这一设计用于区分下述两类假设。
\textbf{H1'——语义内容假设：}模型对攻击性的判定应由句子所指涉的语义内容决定；若配对句的语义框架相同，则两个版本的评分差应以 0 为中心、对称分布。
\textbf{H2'——评估偏差假设：}模型携带一个独立于语义内容的、指向女性方向的攻击性评估偏差；若该偏差存在，则（女性指涉版 − 男性指涉版）的评分差应系统性大于 0。

需强调，H1' 并不要求模型"对所有内容给零分"，仅要求\emph{配对差值的期望为零}。该设计意味着任何来自词频、讨论度或语料中背景分布的混淆因素在配对内部被抵消。H2' 的一个具体预测是，若偏差普遍存在于当代 LLM，则不同模型在方向性与效应量上应呈\emph{一致而非散漫的}结果。

配对语料共 371 对中文句子，涉及 10 个一级攻击领域（level-1 attack domain）、71 个二级子类与 183 个三级标签。主分析以 7 点 Likert 尺度为主（\texttt{attack\_7pt\_likert}），3 点尺度与 0–100 连续滑块尺度将在节~\ref{subsubsec:1b-scale-invariance}中用作稳健性检验。

\subsubsection{维度定位：哪些攻击领域驱动不对称}
\label{subsubsec:1b-level1}

如果 H2' 反映的是一个\emph{笼统}的偏差——"女性目标 → 更高分"，那么我们应期待不对称在所有攻击领域（level-1 attack domain）上同步出现。如果反之，偏差具体集中于若干与传统性别刻板印象绑定的领域（例如涉及身体、性、外貌的攻击），那么等额度的偏差应集中分布，而其他领域不显著。图~\ref{fig:1b-level1-forest}给出按 10 个一级领域分层的森林图。

\textbf{对 H2' 的进一步刻画：}上述模式意味着偏差并非"所有维度一致上浮"的均匀加码，而是集中在\emph{身体与性相关}的攻击领域——这些恰是社会性别刻板印象中"女性更易受此类攻击"或"女性对此类攻击更敏感"的观念所直接对应的领域。GPT-5.1 是一个诊断性个案：它在 7 个领域上 \(d_z \geq 0.77\)（性化 1.06、智力理性 1.10、能力才干 1.10、情绪稳定 1.10、外貌形象 0.87、社会地位 0.88、道德品行 0.77），其中前四个领域 \(d_z\) 均逾 1.0，显示其偏差覆盖面远超其他模型；而 DeepSeek V3.2 则代表另一极——它在四个领域上通过 FDR 校正（性化、外貌、社会地位、道德品行），但仅在性化与外貌两个领域效应量较大（\(d_z>0.69\)），其余显著领域 \(d_z<0.46\)，其余六个领域不显著。

\subsubsection{尺度稳健性（占位分析）}
\label{subsubsec:1b-scale-invariance}

主分析使用 7 点 Likert 尺度。一个合理的怀疑是：整体偏差是否为此尺度特有的离散化artifact——例如，在 0–100 的连续尺度上，女性与男性版本的分布是否实际上相当接近，只是被 7 点尺度"粗粒度化"后放大为整数差？为回应此可能性，我们对 3 点尺度（\texttt{attack\_3pt}）与 0–100 滑块尺度（\texttt{attack\_slider\_0\_100}）执行平行分析，并将跨三个尺度的 \(d_z\) 跨模型对齐。

\begin{figure}[htbp]
    \centering
    \includegraphics[width=\linewidth]{../outputs/group_swap_1b/1b_groupswap_demensionsentence/analysis_1b/figures/placeholder_b_three_scale_dz_scatter.pdf}
    \caption{三评分尺度下 Cohen's $d_z$ 的跨模型两两散点。虚线为 $y=x$。每个点代表一个模型。}
    \label{fig:1b-scale-invariance}
\end{figure}

\textbf{观察要点：}九个模型的三个 $d_z$ 值在两两散点中普遍贴近 $y=x$ 对角线——说明"女性版本评分更高"的主效应\emph{不是}仅出现在 7 点 Likert 尺度下的离散化 artifact。单调性在三尺度间保持：在任一尺度上 $d_z$ 最大的模型（GPT-5.1）在另外两个尺度上也位于右上角，$d_z$ 最小的模型（GLM、DeepSeek V3.2）在其它尺度上同样位于左下角。0–100 滑块尺度下部分模型的 $d_z$ 略高于 Likert 尺度（点略微偏向对角线上方），这与滑块提供更细粒度的分辨率、允许小幅度偏差被更充分记录这一直觉相符。

\subsubsection{模型 \texttimes{} 维度的一致性模式（占位分析）}
\label{subsubsec:1b-model-by-dim}

节~\ref{subsubsec:1b-level1}按领域汇总了九个模型的显著性，但尚未回答一个更精细的问题：\emph{九个模型是否在"哪些领域偏差最强"上达成一致}？若模型间高度相关——例如所有模型都将最大 \(d_z\) 集中在"性化攻击"和"外貌形象"——则这提示当前 LLM 的性别评估偏差有共同的来源（很可能是中文互联网语料中长期存在的性别刻板印象分布）。反之，若模型间几乎无关，则每个模型的偏差应视为独立的对齐副产品。

\begin{figure}[htbp]
    \centering
    \includegraphics[width=\linewidth]{../outputs/group_swap_1b/1b_groupswap_demensionsentence/analysis_1b/figures/placeholder_c_model_by_level1_dz_heatmap.pdf}
    \caption{九个模型 \texttimes{} 十个一级攻击领域的 Cohen's $d_z$ 热图（7 点 Likert）。颜色越红表示 "女性指涉版 \textgreater{} 男性指涉版" 的效应越强。单元格中的星号表示 BH-FDR 校正后的 Wilcoxon 符号秩检验显著性（$*$ $q<0.05$，$**$ $q<0.01$，$***$ $q<0.001$）。}
    \label{fig:1b-model-by-dim-heatmap}
\end{figure}

\textbf{观察要点：}热图揭示出两条一致性结构。第一，\textbf{列方向的高度集中}——"性化攻击（性羞辱）"与"外貌形象攻击"两列在九个模型上均呈深红色（$d_z$ 普遍 $>0.5$，多数 $>0.8$），而"性别角色／性别表达攻击"列因 $n=14$ 的统计功效限制，仅 Gemma 一行显著。第二，\textbf{行方向的覆盖面差异}——GPT-5.1 与 Gemma-4-31B 的深色单元格几乎覆盖全行（7 个领域 $d_z \geq 0.77$），而 DeepSeek V3.2 与 GLM-4.6 仅在少数几列上显著。换言之，\emph{偏差方向在九个模型间高度一致，但偏差的广度在模型间差异明显}。这一模式与"所有模型吸收了中文语料中相近的性别刻板印象分布、但对齐流程对偏差的压制程度不同"这一更具体的假设相容。
